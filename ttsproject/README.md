# Text-to-Speech Web Application

A full-stack Text-to-Speech (TTS) web application that converts written text into spoken audio.

## Features
- Convert text to speech
- Change speech speed
- Voice selection
- Save text history
- View saved history

## Technologies
Frontend: HTML, CSS, JavaScript  
Backend: Java, Spring Boot  
Database: MySQL

## How to Run
1. Clone repository
2. Create database `ttsproject`
3. Run TtsprojectApplication.java
4. Open http://localhost:8080
